5 Lamda Architecture Resources

December 30, 2014

Lambda Archiecture is fascinating Architectural Pattern. I first read about it while researching Apache Storm. Nathan Marz, the creator of Storm, discusses Lamda Archicture in his book Big Data: Principles and best practices of scalable realtime data systems. Here are some good resources for learning more.

  1. Runaway Complexity in Big Data and a Plan to Stop It Nathan Marz great slide deck on reducing complexity in Big Data.

  2. Nathan Marz on Storm, Immutability in the Lambda Architecture, Clojure Nathan Marz Q&A from QCon 2014.

  3. Lambda Architecture: Design Simpler, Resilient, Maintainable and Scalable Big Data Solutions A great primer complete with working source code examples.

  4. Implementing the Lambda Architecture with Spring XD A nice presentation from SpringOne 2014 with multiple example implementations.

  5. Questions About the Lambda Architecture An interesting article summarizing some of the concerns about Lamda Architecture.
